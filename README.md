  ## Description
  This app is a text editor that can be used to edit code snippets, on or offline. 

  ## Technonology
  This app utitlizes the following:
  - HTML
  - CSS
  - Javascript
  - Node.js
    - Express
    - Nodemon
  - Webpackages (HTML Webpackage Plugin)
  - Heroku

  ## Installation
  Install Node.js, then clone this github repository.

  ## Usage
  The deployed application can be found here: https://text-editor-dp.herokuapp.com/
